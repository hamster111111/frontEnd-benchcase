和原论文不一样的地方

这些差异决定了哪些数字不能横向声称超过论文模型。

生成框架不同论文用 Claude Code / OpenHands；我们主要是 direct-HTML scaffold，Level 2/3 没有完整 agent workflow。

DSR 口径不同论文 DSR 是容器 start.sh 部署成功率；我们的 DSR-like 是 HTML 可渲染率或完整率。
Level 2/3 缺 FS我们没有全量执行 workflow functional verification，因此不能给官方 FS/Avg。

评测没有完全可信跑完原始 UI 的 0.8% 包含大量 APIConnectionError；rejudge 只覆盖 195/647，不是全量官方 accuracy。

分母不同论文称 127 pages / 374 interactions；本地 full_dataset 展开为 375 条 records。
